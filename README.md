# Rediseño Web con Enfoque en Usabilidad y Accesibilidad (WCAG)

## Descripción
Proyecto de rediseño web utilizando el framework **Bootstrap 5**. Se reestructuró el sitio del primer bimestre aplicando principios de diseño centrado en el usuario, diseño responsive y accesibilidad web.

## Framework Utilizado
* **Bootstrap 5** (vía CDN)

## Mejoras de Accesibilidad Implementadas
- Estructura semántica en HTML5 (`<header>`, `<nav>`, `<main>`, `<footer>`).
- Contraste de texto y fondo adaptado a estándares WCAG.
- Atributos `alt` en todas las imágenes.
- Menú de navegación responsive adaptado a móviles y escritorio.
- Formulario con etiquetas `<label>` vinculadas correctamente a cada campo de entrada.
