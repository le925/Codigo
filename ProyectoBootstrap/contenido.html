Contenido del curso

Conoce a detalle el temario y los objetivos de cada tecnología para desarrollar sitios web modernos.

Importante: Este temario está diseñado siguiendo los estándares de desarrollo con Bootstrap 5, buenas prácticas de SEO y accesibilidad web.

Tecnologías principales

HTML5 es el lenguaje fundamental sobre el que se construye toda la web. Permite estructurar correctamente el contenido de una página utilizando etiquetas semánticas que facilitan la lectura del sitio tanto a navegadores como a motores de búsqueda (SEO) y lectores de pantalla.

Temas principales del módulo:
  • Estructura básica de un documento HTML5 (<!DOCTYPE html>, <head>, <body>).
  • Etiquetas semánticas: <header>, <nav>, <main>, <section>, <article> y <footer>.
  • Manejo de elementos de texto, enlaces, imágenes y listas.
  • Creación de formularios web accesibles y validación nativa de campos.
  • Inserción de contenido multimedia (audio y video HTML5).
Detalles del nivel:

Nivel: Básico / Principiante

Duración estimada: 12 horas de estudio

Proyecto práctico: Estructurar el maquetado completo de un portal de noticias o blog personal.

CSS3 y Bootstrap 5 te permiten darle estilo, color y un aspecto profesional a tus maquetas HTML. Con Bootstrap, puedes acelerar el proceso de desarrollo utilizando un sistema de rejillas adaptable y componentes listos para usar en móviles, tablets y computadoras.

Temas principales del módulo:
  • Fundamentos de CSS: selectores, modelo de caja (*box model*) y especificidad.
  • Diseños flexibles y adaptables con Flexbox y CSS Grid.
  • Sistema de rejilla (*Grid System*) de Bootstrap de 12 columnas.
  • Uso de clases de utilidad de Bootstrap (márgenes, rellenos, colores, alineación).
  • Componentes de interfaz: Navbars, Cards, Modales, Acordeones y Botones.
Detalles del nivel:

Nivel: Intermedio

Duración estimada: 18 horas de estudio

Proyecto práctico: Diseñar una tienda en línea (*e-commerce*) totalmente adaptable a celulares.

JavaScript es el lenguaje de programación que añade interactividad y dinamismo a tus páginas web. Te permite responder a las acciones del usuario, validar datos en tiempo real y conectarte con servidores o APIs externas para actualizar contenido sin recargar la página.

Temas principales del módulo:
  • Sintaxis moderna: variables (let, const), tipos de datos y estructuras de control.
  • Manipulación del DOM (Document Object Model) para modificar contenido y estilos dinámicamente.
  • Escucha de eventos (clicks, envíos de formulario, tecleo).
  • Funciones tradicionales y *Arrow Functions*.
  • Consumo de datos asíncronos mediante fetch() y Promises.
Detalles del nivel:

Nivel: Avanzado

Duración estimada: 25 horas de estudio

Proyecto práctico: Desarrollar una aplicación web interactiva que consuma datos reales de clima o películas.
